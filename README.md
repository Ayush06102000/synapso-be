







### Installation

1. **Clone the Repository:**
    ```bash
    git clone 
    cd 
    ```

2. **Install dependicies:**
    ```bash
    npm install
    ```

3. **Set up Enviroment Variables:**
 Create a `.env` file in the root directory or change the .env.example to .env 






 5. **Start the server:**
 ```bash
    npm start
    ```
